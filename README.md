# BASIC Programming Language

A tiny interpreter for a BASIC-like calculator language written in Python.

## Features

- Tokenizes and evaluates basic arithmetic expressions
- Supports integers, floats, parentheses, and the operators `+`, `-`, `*`, and `/`
- Includes a simple interactive shell for trying expressions from the terminal

## Run It

Start the REPL with:

```bash
python shell.py
```

Then enter an expression such as:

```text
basic > (2 + 3) * 4
```
